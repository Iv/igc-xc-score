igc-xc-score Demo

igc-xc-score is a LGPLed JS library for scoring paragliding/hang-gliding XC flights according to a configurable set of rules

It can run in the browser, in a desktop application or as a standalone command-line tool. Currently this demo supports only the FFVL rules.

The full source of the library and this webpage are available here:

https://www.github.com/mmomtchev/igc-xc-score

Your IGC file never leaves your computer. You can download this page from the github link and use it while offline. The CPU used for the optimization process is your CPU through the JS engine of your browser. You need a good computer with a good browser. Chrome, Firefox, Safari and the latest Edge since January 2020 should be ok. Everything else is probably not and will be slow.
